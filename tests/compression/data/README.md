# `tests/compression/data/`

Frozen reference artifacts for the `compression-consumes-clustered-dictionary`
differential regression test.

## `epoch_result_reference.json`

Frozen `EpochReport.to_json()` output captured on the pre-refactor
`EpochCompressor.run` pipeline against the deterministic fixture
defined in `tests/compression/_clustered_fixture.py`.

The differential test
(`tests/compression/test_epoch_clustered_consume.py`) re-runs the
same fixture on the post-refactor pipeline and asserts deterministic-
field equality. Three top-level fields are excluded from the
comparison because they vary across runs even when the compression
logic is identical:

- `wall_seconds` — wall-clock; intrinsically non-deterministic.
- `output_checkpoint`, `source_checkpoint` — tempfile paths created
  with random suffixes by `tempfile.TemporaryDirectory`.

Also excluded: per-iteration `validation_report_paths` (empty in
the v0 path, but path-typed and so subject to the same hygiene).

Everything else — convergence reason, iteration sequence, panels,
features zeroed, cross-entropy deltas, sha256 hashes of source +
output safetensors, n_features_zeroed_total — is checked bit-exact.

## Regeneration

The reference is regenerated by running the capture script that
predates the refactor; the script is preserved at
`/tmp/capture_epoch_reference.py` in the development branch but
not committed (one-shot helper).

If the test environment's `polygram` version changes in a way that
affects deterministic compression output (e.g., a new compression
algorithm lands), the reference must be regenerated and the change
landed alongside the impl.

To regenerate manually:

```python
from pathlib import Path
import tempfile
from unittest.mock import patch
from polygram import EpochCompressor
from tests.compression._clustered_fixture import (
    CANONICAL_PROMPTS, EPOCH_KWARGS, build_synth_sae,
    make_synth_prepass_patch,
)

with tempfile.TemporaryDirectory() as tmp:
    sae = build_synth_sae(Path(tmp) / "sae.safetensors")
    epoch = EpochCompressor(sae_checkpoint=sae, prompts=CANONICAL_PROMPTS, **EPOCH_KWARGS)
    with patch(
        "polygram.compression.epoch._compute_firing_rates_and_residuals",
        new=make_synth_prepass_patch(),
    ):
        result = epoch.run(Path(tmp) / "out.safetensors")
    Path("tests/compression/data/epoch_result_reference.json").write_text(
        result.report.to_json()
    )
```
